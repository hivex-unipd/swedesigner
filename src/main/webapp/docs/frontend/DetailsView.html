Class: DetailsView

DetailsView

DetailsView shows the details of an element in one diagram. The diagram can be the main class diagram or an activity diagram. In the first case, the elements are classes or links between them: DetailsView shows the class name, its attributes and methods, etcetera. In the second case, the elements are instruction blocks: DetailsView shows the operands, parameters, or conditions of a block.
